**Группировка по типам ОПФ**

**Некоммерческие организации**

- **Автономная некоммерческая организация (АНО)** - код 70400
- **Частное учреждение (ЧУ)** - код 75404

_Доступны все уровни образования_

**Коммерческие организации**

- **Общество с ограниченной ответственностью (ООО)** - код 12165
- **Акционерное общество (АО)** - код 12247

_Доступны только дополнительные виды образования_

**Индивидуальные предприниматели**

- **ИП** - код 50102

_Доступны только дополнительные виды образования_

**Примечания по использованию**

\- Наличие подходящей ОПФ и ОКВЭД является **необходимым, но не достаточным** условием для получения лицензии  
\- Для получения лицензии организация должна соответствовать всем **лицензионным требованиям** Рособрнадзора  
\- Коды ОКОПФ и ОКВЭД должны соответствовать актуальным версиям классификаторов  
\- Данные актуальны на 2024 год
